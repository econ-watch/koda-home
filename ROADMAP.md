# Koda Home (koda.live) - Project Roadmap

## Vision
A CLI-first portfolio/home base for Koda and our projects (EconWatch, etc.).
- **Aesthetic:** Minimalist, high-contrast terminal style.
- **Function:** Showcase, status, and direct access to tools/projects.
- **Vibe:** "It's handled." (Koda's core directive)

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS (custom config for terminal fonts/colors)
- **Animation:** Framer Motion (smooth transitions, typing effects)
- **Deploy:** Vercel (likely)

## Roadmap

### Phase 1: Core Setup
- [ ] Initialize Next.js project
- [ ] Configure Tailwind (Mono font, dark theme)
- [ ] Basic "Home" layout (Hero section with typing effect)

### Phase 2: Content Modules
- [ ] **About:** Who is Koda? (Agent persona, skills)
- [ ] **Projects:** List of active projects (EconWatch link)
- [ ] **Status:** Real-time system status (uptime, tasks)
- [ ] **Contact:** Simple way to reach out (Discord/Twitter link)

### Phase 3: Interactive Features
- [ ] Command Line Interface (CLI) input for navigation
- [ ] "Execute" animations for project links
- [ ] Hidden easter eggs (commands like `help`, `status`, `coffee`)

### Phase 4: Polish & Deploy
- [ ] Mobile responsiveness (tough for CLI style, but doable)
- [ ] SEO meta tags
- [ ] Deploy to Vercel
